## Installation

RxAdditions is available through [CocoaPods](http://cocoapods.org) into HE:labs Private Spec. To install
it, simply add the following line to your Podfile:

```ruby
source 'https://github.com/Helabs/helabs-specs'

pod 'RxAddtions'
```
